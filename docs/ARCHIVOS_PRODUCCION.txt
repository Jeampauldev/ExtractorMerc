🔧 CONFIGURACIÓN REQUERIDA:
===========================

1. Crear archivo .env basado en .env.template
2. Configurar credenciales Mercurio (Afinia y Aire)
3. Configurar conexión RDS AWS
4. Configurar acceso S3 AWS
5. Instalar dependencias: pip install -r requirements.txt
6. Instalar navegadores: playwright install chromium
